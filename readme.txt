A Wordpress plugin to insert and manage SmartAsset widgets on your blog.

== Description ==

* Insert SmartAsset Captivate widgets in your blog posts automatically or by using a simple shortcode: [sa_captivate]
     
== Installation ==

= Manually =

* Download the Plugin
* Go in your Dashboard to 'Plugins' -> 'Install'
* Click on Upload
* Click on 'Choose file' and choose the downloaded Zip-file
* Click on 'Install'
* Activate the plugin through the 'Plugins' menu in WordPress
* Now you can go to the Settings-Page on 'Settings' -> 'SmartAsset Captivate'
